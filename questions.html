Qu'est-ce qu'un protocol ?

Un protocol est une méthode de communication normée qui permet la communication entre plusieurs périphérique/appareils/applications en informatique.

Ref :Nicov

Qu'est-ce qu'un port ?

Un système permettant de recevoir ou d'emettre des informations.

Ref :Nicov

Quel est le port par défaut du protocol HTTP?

Le port par défaut est 80.

Ref :Nicov

Qu'est-ce qu'HTML ?

HTML qui signifie HyperText Markup Language est un language permettant la création de page web statiques.

Ref :Nicov

Qu'est-ce que CSS ?

CSS est un language facilitant le design d'une page HTML.

Ref :Nicov

Qu'est-ce que JavaScript ?

JavaScript est un language permettant de dynamiser une page web coté client.

Ref :Nicov

Qu'est-ce que PHP ?

Le PHP est un language serveur permettant de dynamiser l'affichage d'une page HTML.

Ref :Nicov

Qu'est-ce que Git ?

git est un outil de partage de fichier permettant de travailler a plusieurs sur un meme projet.
